# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim. The full text is available at the link.

## Reporting

Report unacceptable behavior to the project maintainers at <apollon@aet.cit.tum.de>. All reports are handled confidentially.

For security vulnerabilities, follow the disclosure process in [`SECURITY.md`](./SECURITY.md) instead — security issues are tracked separately from conduct violations.
