# @tumaet/webapp

Per-version bullet log generated by [Changesets](https://github.com/changesets/changesets); the narrative form lives in each [GitHub Release](https://github.com/ls1intum/Apollon/releases). Released in lockstep with `@tumaet/server`.

## 4.4.1

### Patch Changes

- [#683](https://github.com/ls1intum/Apollon/pull/683) Sharing a diagram a second time, or starting a new diagram during a collaboration session, no longer leaves a blank canvas behind.
- [#682](https://github.com/ls1intum/Apollon/pull/682) PNG, SVG, PDF, and JSON export work again on iOS and Android. Closes [#553](https://github.com/ls1intum/Apollon/issues/553).

## 4.4.0

### Major Changes

- [#657](https://github.com/ls1intum/Apollon/pull/657) Deployment break: requires Redis Stack 7.4 and an `OWNER_SECRET` ≥ 32 chars. Diagram keys move from `STRING` to `RedisJSON`; a one-shot migration must run before deploy. Runbook in the [GitHub Release](https://github.com/ls1intum/Apollon/releases/tag/v4.4.0) and in [PR #657](https://github.com/ls1intum/Apollon/pull/657).

### Minor Changes

- [#658](https://github.com/ls1intum/Apollon/pull/658) Export diagrams as animatable PowerPoint slides — every class, arrow, label, and attribute becomes its own native PowerPoint shape, ready for per-element animations.
- [#663](https://github.com/ls1intum/Apollon/pull/663) Live cursors and presence during a collaboration session.
- [#657](https://github.com/ls1intum/Apollon/pull/657) Figma-style version history: named milestones, 30-minute auto-snapshots, preview, one-click restore with undo, permalinks.
