The diagram dashboard has a cleaner, more modern look — a slimmer top bar, lighter shadows, cooler surfaces, and rounded cards — with stronger text and border contrast for better readability and accessibility in both light and dark themes. The top bar is now the same height with consistent buttons on the dashboard and inside the editor, the editor toolbar collapses long labels (like "All diagrams" and "Version history") to icons when space is tight, and dark mode uses one consistent cool palette throughout.
