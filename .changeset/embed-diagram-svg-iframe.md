Embed your shared diagrams anywhere. Drop a diagram into a GitHub or GitLab README, issue, or pull request as a Markdown image — it renders inline and stays in sync as you keep editing — or paste a ready-made iframe snippet into Notion, Confluence, GitLab Pages, or a VS Code preview. The share dialog now offers an "Embed in GitHub / GitLab" panel with copy-paste snippets, and embedding needs no sign-in or tokens: the diagram link is the key, exactly like opening it in the editor. Embeds are served fast and cache automatically, so a diagram that lots of people view stays responsive.
